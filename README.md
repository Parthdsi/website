# Parth's Website

A personal portfolio built with React + Vite. Deploys to parthdsi.xyz/ (root).

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

Pushes to `main` trigger a deploy. Enable it once: **Settings → Pages → Build and deployment → Source: GitHub Actions**.

To use parthdsi.xyz at the root: add it as the custom domain in **Settings → Pages**.
